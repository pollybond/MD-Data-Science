# Weather Forecast Project

Проект по классификации климатических зон и среднесрочному прогнозированию температуры на основе метеоданных шести городов России.

## Установка и запуск

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Подготовьте данные:**
   Поместите `.parquet` файлы с метеоданными в папку `data/`.

3. **Запустите ноутбук:**
   ```bash
   jupyter notebook Классификация временных рядов и их прогнозирование.ipynb
   ```

## Ключевые особенности

*   **ROCKET для классификации временных рядов** — сохраняет временную структуру и фазовые сдвиги.
*   **Агрегированные признаки для LightGBM** — решает проблему высокой размерности и ускоряет обучение.
*   **Адаптация под климат** — различные гиперпараметры моделей для субтропических и муссонных зон.
*   **Хронологический сплит данных** — строго предотвращает утечку будущих данных (data leakage).

## Ограничения

*   Точность прогноза на 30 дней ограничена фундаментальной хаотичностью атмосферных процессов.
*   Переходные сезоны (весна/осень) временно снижают точность классификации климатических зон.
*   Для дальнейшего улучшения метрик требуется интеграция внешних признаков (спутниковые данные, атмосферное давление, данные соседних станций).
